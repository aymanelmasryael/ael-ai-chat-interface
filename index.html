About AEL AI Chat

A unified chat interface by AEL Digital Studio

📋 Overview

AEL AI Chat Interface provides a unified, consistent chat experience across multiple AI providers including OpenAI (GPT-4, GPT-3.5) and Anthropic (Claude). Built with vanilla JavaScript, it offers full control over your conversations without vendor lock-in.

⚡ Features

  • Multi-model support (OpenAI + Anthropic + Demo)
  • Conversation history with localStorage persistence
  • Custom system prompts with presets
  • Adjustable temperature and max tokens
  • Export conversations as JSON
  • Glassmorphism UI with dark theme
  • Zero external dependencies

🛠 Tech Stack

  • HTML5 — Semantic structure
  • CSS3 — Glassmorphism, custom properties, flexbox/grid
  • JavaScript — Vanilla JS, Fetch API, localStorage
  • APIs — OpenAI REST API, Anthropic REST API
